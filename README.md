# Nettside for NODE

- Linjeforeningen for Kunstig Intelligens ved Universitetet i Bergen

## Komme i gang

1. Installer avhengigheter:
```bash
   npm install
```

2. Kjør utviklingsserver:
```bash
   npm run dev
```

3. Åpne [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
